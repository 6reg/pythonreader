For-loop

A for-loop repeats code for a specified number of times. For example, here's the code you could write to print "Python rocks socks!" 100 times: 

 for i in range(100): 
   print("Python rocks socks!")

Comparison Operators

To compare values in python, we can use comparison operators .

Here are the comprison operators:

Operator Meaning Example Value
== equals 1 + 1 == 2 True
!= does not equal 3.2 != 2.5 True
< less than 10 < 5 False
> greater than 10 > 5 True
<= less than or equal to 126 <= 100 False
>= greater than or equal to 5.0 >= 5.0 True

We can use comparison operators to make conditions for if statements and while loops

For example: 

if 1 < 2: 
    print("1 is less than 2")

Here's another example: 

num = int(input("Enter a number: "))
if num == 0: 
   print("That number is 0")
else: 
   print("That number is not 0")

Logical Operators

Logical operators can be used to change or combine boolean statements.

Here are the logical operators:

Operator Example Result
not not (2 == 3) True
and (2 ==3) and (-1 < 5) False
or (2 == 3) or (-1 < 5) True
